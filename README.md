Краткое описание
Необходимо написать программу под Windows, которая обеспечит удобство выбора частот для программирования раций и систематизацую используемых и использованных частот.

Исходные данные
- Есть диапазон частот 400-470 МГц.
- Диапазон выше равномерно разбивается на 16 поддиапазонов. То есть ширина каждого поддиапазона примерно = 70/16 = 4.375 МГц.
- Между каждым поддиапазоном должна быть неиспользуемая часть(бланк) шириной 0.02 МГц, то есть по факту ширина каждого диапазона будет около = 4.375 - 0.02 = 4.355 МГц.
- Каждый поддиапазон будет содержать равноудаленные каналы, ширина между которыми будет 0.003 МГц. То есть в одном поддиапазоне будет около = 4.355/0.003 ~= 1450 каналов.

Что программа должна делать
Программа должна сгенерировать 16ть столбцов(соответсвующих 16ти поддиапазонам) и примерно 1450 строк(соответствующих количеству каналов в каждом поддиапазоне). Каждый канал в каждом столбце должен иметь не больше трех знаков после запятой(например 433.333, а не 433.3333). Каналов в каждом поддиапазоне должно быть ~1450, как и строк и они должны расположнены в случайном(псевдослучайном) порядке, чтобы нельзя было легко предугадать значение следующего канала в строке.
